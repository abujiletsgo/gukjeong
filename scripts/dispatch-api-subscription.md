# 국정투명 — API 구독 자동화 지침 (Claude Dispatch용)

## 목표

대한민국 시민 투명성 플랫폼 "국정투명"에 필요한 모든 공공데이터 API에 구독 신청합니다.
총 3개 포털에서 API 키를 발급받아야 합니다.

---

## PART 1: data.go.kr (공공데이터포털) — 대부분의 API가 여기 있음

### 1-1. 로그인

1. https://www.data.go.kr 접속
2. 우측 상단 "로그인" 클릭
3. 이미 로그인되어 있으면 건너뛰기
4. 로그인이 안 되어 있으면 사용자에게 직접 로그인하도록 요청 (비밀번호 입력 필요)

### 1-3. 각 API 구독 절차 (반복)

각 API에 대해 다음을 수행합니다:

1. data.go.kr 상단 검색창에 API 이름을 입력하고 검색
2. 검색 결과에서 가장 일치하는 "오픈API" 항목 클릭
   - "오픈API" 탭이 따로 있으면 그것을 클릭
   - "파일데이터"가 아닌 "오픈API"를 선택해야 함
3. 상세 페이지에서 "활용신청" 버튼 클릭
4. 활용신청 양식이 나타나면:
   - **활용목적**: `시민 투명성 플랫폼(국정투명) 개발을 위한 공공데이터 활용. 예산, 감사, 국회 활동 등 공공 정보를 시각화하여 시민에게 제공하는 비영리 목적의 웹서비스입니다.`
   - **활용유형**: "웹 서비스 개발" 또는 가장 적합한 항목 선택
   - **트래픽**: 개발 단계이므로 기본값(1,000건/일) 유지
   - **동의 체크박스**: 모두 체크
5. "신청" 또는 "제출" 버튼 클릭
6. 승인 대기 또는 즉시 승인 확인

**참고:**
- 일부 API는 이름이 정확히 일치하지 않을 수 있음. 핵심 키워드로 검색 (예: "분야별 예산", "수의계약", "감사결과")
- "해당 API를 찾을 수 없음" → 유사한 이름의 API가 있으면 그것을 신청
- 이미 신청한 API는 "활용신청" 대신 "활용중" 표시됨 → 건너뛰기
- 검색 결과가 여러 개면 제공기관이 위 목록의 기관과 일치하는 것을 선택

### 1-4. API 키 확인

모든 구독 완료 후:
1. data.go.kr → 마이페이지 → 활용신청 현황
2. 일반 인증키(Encoding) 값을 복사
3. 이 키 하나가 위 모든 API에 공통으로 사용됨

## PART 2: 국가법령정보 (open.law.go.kr)

1. https://open.law.go.kr 접속
2. 회원가입 또는 로그인
3. "오픈API" 또는 "API 이용안내" 메뉴 찾기
4. API 키 발급 신청
   - 활용목적: `시민 투명성 플랫폼(국정투명)에서 법률안 상세 정보와 법령 전문을 제공하기 위한 공공데이터 활용`
5. 발급된 API 키 기록

## PART 3: 빅카인즈 (bigkinds.or.kr)

1. https://www.bigkinds.or.kr 접속
2. 회원가입 또는 로그인
3. "뉴스분석 API" 또는 "API 신청" 메뉴 찾기
4. API 이용 신청
   - 이용목적: `시민 투명성 플랫폼(국정투명)에서 뉴스 프레임 분석 및 언론 보도 다양성 시각화를 위한 비영리 연구 목적 활용`
   - **참고**: 빅카인즈는 연구/공익 목적 심사가 있어 승인까지 시간이 걸릴 수 있음
5. 발급된 API 키 기록
