# 조달 맥락 분류 체계 (Procurement Context Classification)

AI 감사 시스템이 의심 패턴을 탐지한 후, 실제 비리 여부를 판단하기 위한 맥락 분류 체계.
**기본 원칙: 무죄 추정 — 합리적 설명이 있으면 비리가 아님.**

## 핵심 법적 근거: 국가계약법 시행령 §26 (수의계약)

| 조항 | 사유 | 적용 분야 |
|---|---|---|
| §26①2 | 국가안보/기밀 | 국방, 경찰, 정보기관 |
| §26①3 | 특정인의 기술 필요/경쟁 불가 | SW 유지보수, 특허의약품, OEM 장비 |
| §26①3(바) | 조달청 지정 우수조달/혁신제품 | 제3자단가계약, MAS |
| §26①5 | 재공고 후 단독 참여 | 구조적 독점 시장 |
| §26①6 | 소액 (기준금액 이하) | 소규모 계약 |

## 정상 제도 (NORMAL — 자동 제거)

### A. 제3자단가계약 / 우수조달물품 / 혁신제품
- **법적 근거**: 조달사업법 시행령 §30, §32; 국가계약법 시행령 §26①3(바)
- **제도**: 조달청이 품질 인증 + 가격 협상을 완료한 후 지정. 수요기관은 사전 협상 단가로 직접 주문.
- **왜 100% 낙찰**: 예정가격 = 사전 협상 단가이므로 100%는 제도적 설계
- **왜 단독 응찰**: 사전 선정된 공급업체만 참여하므로 경쟁 입찰 개념 아님
- **진짜 의심**: 조달청 지정 과정 자체의 평가위원 매수, 불필요한 대량 구매, 더 싼 대안 존재 시

### B. 지역 농산물/임산물 조달
- **법적 근거**: 농어업경영체법, 학교급식법 (지역 우선 조달)
- **제도**: 지역 농협/산림조합 통한 직접 구매, 공공급식통합플랫폼(eat.co.kr)
- **진짜 의심**: 시장가 대비 현저히 고가

## 낮은 위험 (LOW_RISK — 점수 하향, 맥락 표시)

### C. 관급자재 / 건설자재 (점수 상한: 40)
- **제도**: 시설공사 맞춤형서비스 관급자재 선정 운영기준 (조달청 고시)
- **구조**: 레미콘 — KS F 4009 기준 생산 후 90분 이내 타설 필수 → 반경 30-40km 자연 독점
- **골재**: 경제적 운송 반경 ~50km
- **2024 변경**: 관급철근 MAS(다수공급자계약) 공급 전환 시작
- **진짜 의심**: 지역 레미콘 업체 간 가격 담합 (공정위 적발 사례: 은평-파주 19개사 131억 과징금, 광양 7개사 22억, 천안-아산 18개사 7억)

### D. 유지보수 / SW / 라이선스 (점수 상한: 35)
- **법적 근거**: 국가계약법 시행령 §26①3; 소프트웨어사업 계약 지침 (과기정통부); 소프트웨어진흥법 §46
- **구조**: 원개발사만 소스코드 접근 가능 + 저작권법상 수정 권한 → 기술적 독점
- **업계 표준**: 유지보수비 = 초기 개발비의 10-15%/년, 외산 SW(Oracle, SAP) 15-22%/년
- **진짜 의심**: 유지보수비 > 개발비의 20%/년, 원개발사가 100% 재하도급, 10년+ 대안 검토 없이 독점, 연간 20%+ 금액 증가

### E. 교육: 교과서/급식/교복 (점수 상한: 30)
- **교과서**: 교과용도서에 관한 규정 → 한국교과서협회 → 출판사별 지역 공급소(1인 사업자)가 독점 유통. 물류는 파주 물류센터가 담당.
- **급식**: 학교급식법 → 지역 소규모 식자재 업체 직납. 종업원 0-1명은 구조적 특성.
- **교복**: 학교별 지정 업체 (최근 가격 담합 조사 있었음)
- **유령업체 오탐**: 교과서 공급소가 0-1명인 것은 정상 — ghost_company에서 제외 필요
- **진짜 의심**: 교복 가격 담합, 급식업체-학교 관리자 인척 관계

### F. 국방/소방/경찰 (점수 상한: 35)
- **법적 근거**: 방위사업법 §35 (방산업체 지정), 국가계약법 시행령 §26①2 (국가안보)
- **구조**: 방산업체 = 산업부 지정 필수, D2B(국방전자조달시스템) 별도 운영, 소방장비 = KFI 인증
- **소방차**: MAS 계약 4종 + 경쟁입찰 8종만 존재 → 자연 제한
- **진짜 의심**: 비기밀 물품(사무용품 등)을 보안 명목 수의계약, 퇴직 공무원의 방산업체 취업 후 수주 (전관예우/방산비리)

### G. 의약품/의료장비 (점수 상한: 35)
- **법적 근거**: 약사법 §31 (특허의약품), 의료기기법
- **구조**: 특허 만료 전 오리지널 제약사만 공급 가능, OEM 의료기기(Siemens, GE, Philips)는 전용 서비스 SW + 부품
- **가격 규제**: 건강보험 실거래가 상환제로 가격 상한 존재
- **공정위 사례**: 지멘스 의료기기 유지보수 독점 남용 48억 과징금
- **진짜 의심**: 제네릭 존재 시 오리지널 고집, 유지보수비 > 장비가의 20%/년, 비OEM 업체에 유지보수 발주 (프론트 가능성)

## 100% 낙찰률의 올바른 해석

| 참여업체 수 | 낙찰률 | 해석 | 조치 |
|---|---|---|---|
| 1개사 | 100% | 수의/협상계약 (가격 사전 결정) | **정상 — 제거** |
| 1개사 | 99.5-99.9% | 가격 추정 능력 우수 또는 사전 정보 | 반복 시 의심 |
| 2개사+ | 100% | 전원이 예정가격을 정확히 맞춤 | **예정가격 유출 강력 의심** |
| 2개사+ | 99.5-99.9% | 통계적으로 매우 이례적 | 5회+ 반복 시 의심 |

## 진짜 의심해야 할 것 (HIGH CONFIDENCE)

1. **복수업체 100% 낙찰률** — 전원이 예정가격을 맞추는 것은 정보 유출 없이 불가능
2. **같은 업체 조합 반복 입찰 참여** — 들러리 입찰 (공정위 적발 패턴 #1)
3. **계약 후 30%+ 금액 증액** — 저가 수주 후 변경계약 수법
4. **동일 주소/대표의 복수 업체** — 위장 경쟁 (공정위: 전체 담합의 25%)
5. **실제 개업일이 계약일 직전** — 특수목적법인 (페이퍼 컴퍼니)
6. **지역 건설자재 가격 균일성** — 레미콘/골재 카르텔 (공정위 반복 적발)
