# 국정투명 데이터 윤리 헌장

## 1. 프라이버시
- 실명 저장 금지
- Kakao/Naver OAuth → 일방향 해시 → UUID
- K-익명성: 설문 집계 그룹 최소 30명
- 개인 설문 응답 외부 비공개

## 2. 공정성
- 모든 정부에 동일 기준 적용
- AI 분석 결과에 "AI 분석" 표시
- 감사 결과는 "의심 패턴"으로만 표현

## 3. 투명성
- 모든 데이터 출처 명시
- 오픈소스 코드 공개
- 미디어 분류 기준 공개

## 4. 법적 준수
- 공공데이터법(2013) 준수
- 저작권법 준수 (뉴스: 제목+링크+AI요약만)
- 명예훼손 방지 ("비리 확정" 표현 금지)
